# SitePulse - Monitor de Sites

Um monitor moderno e profissional de disponibilidade de sites com interface responsiva.

![Status](https://img.shields.io/badge/Status-Production-green)
![License](https://img.shields.io/badge/License-MIT-blue)

## ✨ Funcionalidades

- **Monitoramento em tempo real** de múltiplos sites
- **Gráficos** de tempo de resposta
- **Página de Status** profissional (estilo UptimeRobot)
- **Modo Claro/Escuro**
- **Responsivo** (celular, tablet, desktop)
- **Notificações** quando site cair
- **Export/Import** de dados
- **Deploy automático** via Jenkins + GitHub Pages

## Como Usar

1. Abra `index.html`
2. Adicione os sites que deseja monitorar
3. Acompanhe o desempenho em tempo real
4. Acesse a página `status.html` para visão geral profissional

## Estrutura

- `index.html` - Painel principal
- `status.html` - Página de status
- `js/app.js` - Lógica principal
- `sw.js` - Service Worker (cache offline)

## Deploy

- GitHub Pages: [Link](https://easflab.github.io/sitepulse-monitor)
- Jenkins: Pipeline configurada com deploy automático

## Tecnologias

- HTML5 + Tailwind CSS
- Chart.js
- Vanilla JavaScript
- Service Worker
- Jenkins + GitHub Actions
